# Smart Libraries

A recommendation system built step by step — starting with content-based filtering and expanding toward collaborative approaches.

---

## Overview

This project explores how recommendation systems work under the hood.

The goal is to implement core algorithms from scratch, understand the underlying mathematics, and gradually evolve the system into a more complete hybrid recommendation engine.

The repository will be updated continuously as new components are implemented.

---

## Current Focus

### Phase 1: Content-Based Recommendation System

Using:
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

Goal:
Recommend movies/books based on similarity in genres, keywords, and descriptions.
